Postgresql JDBC Driver 42.3.0 tester

This app compares versions 42.3.0 and 42.2.24 of the postgresql jdbc driver. The version is specified in pom.xml.

       
        <dependency>
            <groupId>org.postgresql</groupId>
            <artifactId>postgresql</artifactId>
            <version>42.3.0</version>
        </dependency>
If the dollar value ends in 4 zeroes and the user repeatedly clicks "edit" then "save" after about 6 times the last 4 zeroes will be truncated.
